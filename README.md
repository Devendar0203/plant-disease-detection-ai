# 🌿 Plant Disease Detection AI

An AI-powered web application that detects plant diseases from leaf images using Deep Learning. This project leverages MobileNetV2 and Streamlit to provide fast, accurate, and user-friendly disease predictions along with treatment suggestions.

---

## 🚀 Overview

Plant Disease Detection AI is designed to help farmers, researchers, and developers identify plant diseases efficiently. By simply uploading an image of a plant leaf, the system predicts the disease class and provides relevant insights.

## ✨ Features

- 📸 Upload or capture plant leaf images
- 🤖 AI-based disease prediction using MobileNetV2
- 📊 Displays prediction with confidence score
- 💊 Provides disease description and treatment suggestions
- ⚡ Fast and interactive UI using Streamlit
